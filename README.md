# Hypixel Bank Tracker
A Python tool to track personal bank history.

## Features
- Fetches data from Hypixel API every 15 minutes.
- Stores transactions in a local SQLite database.
- Visualizes balance trends by using DataGrip's built-in graph features

## Files
- `banktracker2.py`: Main script.
- `bankmanager.db`: Sample database showing successful data collection.
- `bankmanager.csv`: Contains a snapshot of the data collected by this bot. 

## Privacy
This tool is strictly for my own account and does not track other players or de-anonymize anyone.

## Security 
I use a json file to store my API key; it's never hard-coded.
